# WWE 2K24 CAW Models Project

## Introduction
This project focuses on creating Custom Arena Wrestlers (CAWs) for the game WWE 2K24. Each CAW is designed to enhance the gaming experience by providing unique character models that reflect the individualities of various wrestlers.

## Features
- **Detailed Character Models**: Each CAW model is carefully crafted to mimic the appearance of real wrestling superstars.
- **Customization Options**: Users can customize features like attire, physical attributes, and entrance music.
- **User-Friendly Interface**: The project includes tools that simplify the process of creating and modifying CAW models.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mykahrock18-ai/husbenct-me-and-you.git
   ```
2. Navigate to the project directory:
   ```bash
   cd husbenct-me-and-you
   ```
3. Follow the setup instructions to configure your environment for WWE 2K24.

## Usage
- Open the CAW creation tool provided within the project.
- Select a pre-made model or create a new one from scratch.
- Customize the CAW according to your preference.
- Save and export the CAW to use in WWE 2K24.
