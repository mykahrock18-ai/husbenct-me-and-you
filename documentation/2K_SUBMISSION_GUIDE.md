# 2K Community Creations Submission Guide

This guide will help you prepare your CAW (Create a Wrestler) models for submission to the 2K Community Creations website. Please follow the instructions carefully to ensure a successful submission.

## 1. Render Quality Standards

To meet the visual standards required for submission, your renders should:
- **Be high-resolution**: Aim for 1920x1080 or higher.
- **Have professional lighting**: Ensure that the wrestler is well-lit, with no harsh shadows or overexposure.
- **Use a blue background**: The background should be a solid blue, closely matching the reference images provided by 2K.

### Reference Images
Make sure to reference the provided images for examples of acceptable quality and standards.

## 2. ZIP File Structure

Before submitting, package your files in a ZIP archive. Follow this structure:
```
/YourSubmissionName/
  └── Render/
      ├── [YourWrestlerName]_Render1.png
      ├── [YourWrestlerName]_Render2.png
  └── [YourWrestlerName].json
```

- **Render Folder**: Include all high-quality renders in PNG format.
- **JSON File**: The file should contain metadata about your creation. See the metadata template section below.

## 3. Metadata Templates

You should include a JSON file with the following template structure:
```json
{
  "name": "Your Wrestler Name",
  "description": "Description of your wrestler",
  "tags": ["tag1", "tag2"],
  "creator": "Your Name"
}
```
- **name**: The name of your wrestler.
- **description**: A brief description of the character.
- **tags**: Relevant tags to help users find your creation.
- **creator**: Your GitHub username or any name you prefer.

## 4. Step-by-Step Submission Instructions

1. **Prepare your files**: Ensure all images are in the proper resolution and format, and the JSON file meets the specifications.
2. **Create ZIP Archive**: Package your files following the outlined ZIP file structure.
3. **Visit the 2K Community Creations Site**: Go to the submission page.
4. **Upload Your ZIP File**: Select and upload your packaged ZIP file.
5. **Complete any additional information required**: Fill out any required forms related to your submission.
6. **Submit**: Review all the information and submit your creation.

## Conclusion
Following these guidelines will help ensure that your CAW models are presented in the best possible way on the 2K Community Creations website. Good luck with your submissions!
